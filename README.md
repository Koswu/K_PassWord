#K密罐(KPassword)
一款基于HmacSHA256的密码管理APP
****************************************
原理:通过使用一个记忆密码和一个区分代号，经过HmacSHA256哈希函数处理截断后生成16位长的密码

用途:通过使用弱密码来生成强密码，保障账户安全
